Other notable features.

Multiple GitLab Hosts

Trident allows you to add multiple GitLab hosts, so you can easily switch host without re-entering login details.

Variable Font Size

Project files and diff views support variable font size, change the font size to something easier to read.

Supports GitLab 7x and above

Trident has been well tested on GitLab 7x and above. Though, if you see bugs, we're interested in your feedback!

Self Signed Certificate Support

If you don't use a standard Certificate Authority, don't worry! Trident will prompt you to trust a certificate based on the SHA1 fingerprint. Your certificate will then be pinned.

App Transport Security Enabled

Improved security for gitlab.com and github.com connections using Certificate Pinning and TLS 1.2 Forward Secrecy. Trident will force TLS 1.2 minimum on all secure connections.

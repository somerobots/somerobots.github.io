Trident for iOS

A powerful GitLab client for your pocket that has been designed from the ground up for iOS.

Made by Some Robots Ltd

Features

Dashboard and History

All recent project events (for the past three days) can be viewed from the Dashboard. Individual projects have their own history tab, allowing you to go back further in time.

Multiple GitLab Hosts

Trident allows you to add multiple GitLab hosts, so you can easily switch host without re-entering login details.

Issues, Milestones and Labels

Easily create, edit, comment and close/reopen issues. Organise your issues with labels. Track your progress with milestones.

Project Files and Syntax Highlighting

Project files and folders can be easily browsed from a specific branch or commit. Source files support syntax highlighting (we are adding more languages with later releases).

Merge Requests

Compose, view, update, comment and approve merge requests. Dive in the detail and see exactly what's changing with the diff view.

Nightmode and Variable Font Size

Enable nightmode for comfortable reading at night. Change the font size to something easier to read.

Self Signed Certificate Support

If you don't use a standard Certificate Authority, don't worry! Trident will prompt you to trust a certificate based on the SHA1 fingerprint.

App Transport Security Enabled

Improved security for gitlab.com connections using forward secrecy. Trident will force TLS 1.2 minimum on all secure connections.

Supports GitLab 7x and above

Trident has been well tested on GitLab 7x and above. Though, if you see bugs, we're interested in your feedback!
